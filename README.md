Plotly Live Plot
========================i

## Requirement

- nvm (recommended)
- node v.6

## Setup
```sh
$ nvm install v6
$ nvm use v6
$ npm install
```
## Usage

``$ { DATA_SOURCE } > node .``

And open `http://localhost:8080`

## Test Mode

Plot random data for tests

``$ NODE_ENV=test node .``
